# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates hosting a static website using Amazon S3.

## Services Used
- Amazon S3
- Bucket Policy
- Static Website Hosting

## Steps Performed
1. Created S3 bucket
2. Disabled block public access
3. Uploaded index.html
4. Enabled static website hosting
5. Configured bucket policy for public access

 
## Outcome
Successfully deployed a static website using AWS Free Tier.
